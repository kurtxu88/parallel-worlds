# Demo Script

Use this script when recording clips, making short videos, or giving a quick walkthrough.

## 30-Second Version

1. Open the landing page and state the premise:
   Parallel Worlds is an open-source AI storytelling stack that turns short prompts into playable worlds.
2. Jump to create and choose a starter seed.
3. Submit the world and show the generation state.
4. Open the completed story and play one interaction.
5. Close on the stack:
   Vue frontend, FastAPI API, Neo4j for world structure, Postgres for events and settings.

## 2-Minute Version

1. Start on the landing page and explain why the project exists.
2. Show the starter seeds and explain that contributors can use them to verify the system quickly.
3. Create a world and explain the async worker loop.
4. Open the world list and show state changes from queued to ready.
5. Enter the world, play an interaction, and mention event history restore.
6. End on the repo and point to the roadmap and contribution paths.

## Reliable Demo Seeds

- Drowned Moon Archives
- City on a Sleeping Leviathan
- The Orchard That Forgets

## Recording Checklist

- use a clean browser profile
- keep the terminal visible when the worker status matters
- avoid long setup steps in the final cut
- always show one concrete moment of interaction, not only creation screens
- end with the repo page, roadmap, or local stack running
