# Demo 脚本

这份脚本适合录短视频、做演示 clip，或者快速向别人介绍项目。

## 30 秒版本

1. 从 landing page 开始，用一句话说明项目：
   Parallel Worlds 是一个把短 prompt 变成可游玩世界的开源 AI 互动叙事栈。
2. 切到创建页，选一个 starter seed。
3. 提交世界，展示生成状态。
4. 打开已完成的故事，进行一次互动。
5. 最后补一句技术栈：
   Vue 前端、FastAPI API、Neo4j 保存世界结构、Postgres 保存事件和设置。

## 2 分钟版本

1. 从 landing page 开始，解释项目为什么存在。
2. 展示 starter seeds，并说明这让贡献者能更快验证系统。
3. 创建一个世界，解释异步 worker 闭环。
4. 打开世界列表，展示状态从排队到可进入的变化。
5. 进入世界，完成一次互动，并提到历史恢复能力。
6. 最后回到 repo，展示 roadmap 和贡献入口。

## 稳定可复现的 Demo Seed

- 溺月档案馆
- 沉睡巨兽之城
- 遗忘果园

## 录制前检查清单

- 使用干净的浏览器 profile
- 如果要展示 worker 状态，保留终端画面
- 最终成片里尽量避免冗长 setup 步骤
- 一定要展示一次真实互动，不要只停留在创建界面
- 结尾回到 repo 页面、roadmap 或本地运行中的完整栈
