# 行为准则

## 我们的标准

我们希望 Parallel Worlds 是一个让人可以安心学习、构建和协作的项目，而不是一个让人被轻视、被排挤或不敢说话的地方。

有帮助的行为包括：

- 在技术分歧中保持尊重
- 在保持诚实的前提下，优先假设对方出于善意
- 欢迎不同背景和经验水平的贡献者
- 针对代码和想法给反馈，而不是针对人

不可接受的行为包括：

- 骚扰、威胁、歧视
- 人身攻击或侮辱性语言
- 未经允许公开他人的私人信息
- 反复恶意打断讨论、review 或协作流程

## 执行

维护者可以删除、编辑或拒绝违反本行为准则的评论、commit、issue 或其他贡献内容。

## 举报

如果你遇到行为准则相关问题，请通过 [SECURITY.zh-CN.md](./SECURITY.zh-CN.md) 中说明的私下渠道联系维护者。
